# Bangumi.cyou

Bangumi.cyou 是非 Bangumi 官方的公益免费子域名，每个人都可以创建一个属于自己的子域名

例如: 你想要的前缀.bangumi.cyou

# 服务条款
如果您对我们申请了子域名，则代表您同意以下服务条款

1.您的网站不能包含色情内容

2.您的网站不能包含种族主义内容

3.您的网站不能包含歧视特定群体的内容（例如LGBT🏳️‍🌈和Transgender🏳️‍⚧️）

4.Bangumi.cyou 管理组有权因为您的服务存在违规内容而停止解析

# 警告
作为存储库的所有者，您可以完全控制已发布的内容。 但这也意味着发布网站所带来的所有权利和义务仍由您负责。

# 准备好开始使用了？
请在GitHub上的 [issues](https://github.com/everything-cyou/bangumidotcyou/issues) 上通过填写表格进行申请，我们尽快处理您的请求

通过申请的网站将会添加到[列表](https://www.bangumi.cyou/websites)中，您也可以选择不公开
