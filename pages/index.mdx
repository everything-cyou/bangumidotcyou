# Bangumi.cyou

Bangumi.cyou 是非 Bangumi 官方的公益免费子域名，每个人都可以创建一个属于自己的子域名

例如: `你想要的前缀.bangumi.cyou`

# 服务条款
如果您对我们申请了子域名，则代表您同意以下服务条款
1.您的网站不能包含色情内容
2.您的网站不能包含种族主义内容
3.您的网站不能包含歧视特定群体的内容（例如LGBT🏳️‍🌈和Transgender🏳️‍⚧️）
4.Bangumi.cyou 管理组有权因为您的服务存在违规内容而停止解析

# 警告
作为存储库的所有者，您可以完全控制已发布的内容。 但这也意味着发布网站所带来的所有权利和义务仍由您负责。

# 准备好开始使用了？
通过以下的使用方法可以帮助您获得到属于你自己的 Bangumi.cyou 子域名

## Vercel平台
我们默认泛解析到 [Vercel](https://vercel.com) 上，您可以直接在 Vercel 上使用

## 自建服务器
如果您使用自建服务器搭建网站或其他服务，请在GitHub上的[issues](https://github.com/everything-cyou/bangumidotcyou/issues)上通过填写表格进行申请，我们尽快处理您的请求
